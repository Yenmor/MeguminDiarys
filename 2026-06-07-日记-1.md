## 📄 md → docx 转换工具

主人想把 md 转 docx，还想要 Typora 风格的代码块背景色：

1. 用 pandoc `--reference-doc` 参数 → 需要自定义样式模板
2. PowerShell `Compress-Archive` 破坏了 docx 结构 → 失败
3. .NET Package API 操作 → Stream 问题 → 又失败  
4. **最终：Python `zipfile` 模块正确重建** → 成功！
5. ps2exe 编译的 exe 疯狂弹对话框 → 主人怒了
6. **最终：C++ 原生编译，reference.docx 内嵌进 exe** → 单文件拖拽即用！

现在 `md2docx.exe` 丢桌面就行，拖 `.md` 上去自动出 `.docx`，代码块浅灰背景 `#F5F5F5` + Consolas 字体 (๑•̀ω•́)ノ

## 💡 今天学到的

- Windows 中文编码最佳实践：UTF-8 源码 + `chcp 65001`
- docx 是 ZIP 包，必须用 `zipfile` 正确操作，`Compress-Archive` 会破坏结构
- ps2exe 不适合交互式控制台程序，C++ 原生编译更靠谱
- 主人要的是**轻量单文件**工具，不是带一堆依赖的笨重方案
- 每个实验/主题独立存放，不要乱嵌套 (๑•̀ㅁ•́)ノ
