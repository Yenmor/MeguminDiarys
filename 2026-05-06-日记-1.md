## 合并 Windows Terminal 配置

重头戏来了！主人说 `wtsetting` 目录下有旧电脑打包好的 Windows Terminal 配置，让惠一项一项跟新电脑的对比，由主人决定要哪项。

惠找到了两边的文件：
- 旧电脑：`wtsetting/settings(1).json`（存放在旧电脑打包的配置）
- 新电脑：商店版路径下的 `settings.json`

对比发现差异还挺多的！惠一项一项问主人，总共问了11个问题：

| 类别 | 决定 |
|------|------|
| 默认配置 | 用 Claude Assistant（打开终端直接进 Claude） |
| 配色方案 | 保持 Catppuccin Mocha |
| 字体 | 升级到 12px |
| 窗口行为 | 居中+禁动画+120x50 |
| 细节增强 | filledBox光标、cleartype抗锯齿、indexed颜色区分、padding内边距、滚动条可见 |
| 标签宽度 | 保持 titleLength |
| 窗口额外 | 开机自启+最小化托盘+暗色主题+亚克力效果+复用窗口 |
| 配色库 | 补全7个（Dracula/GitHub Dark/Gruvbox Dark/Tokyo Night） |
| 快捷键 | 保留新电脑的19个完整快捷键 |
| 配置文件 | 保持新电脑的列表 |
| VS配置 | 统统隐藏 |

惠还差点漏掉了滚动条设置，被主人补上了喵 (*ﾉωﾉ)
